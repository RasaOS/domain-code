# TASK-001 — Smoother task-management lifecycle

**User story.** As a project lead, I want a rough-draft staging
structure where loose ideas become phases and phases get promoted
into the worked backlog — a smoother end-to-end task flow.

**Why.** The jump from "loose idea" to "task in a phase, ready to
work" is currently ambiguous; an explicit draft → phase → backlog
promotion path makes the lifecycle smooth and predictable.

**Notes (raw idea).** A rough-draft structure to draft up phases
and promote them into the backlog for work.

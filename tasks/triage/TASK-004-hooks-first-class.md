# TASK-004 — Hooks and scripts as first-class enforcement primitives

**User story.** As a kit maintainer, I want hooks and scripts used
wherever appropriate to enforce that things happen correctly — the
way /git-guard does — so results are uniform and unambiguous.

**Why.** Relying on the AI to remember a process is fragile;
deterministic hooks and scripts make the process happen the same
way every time.

**Notes (raw idea).** Use hooks more where appropriate to ensure
things happen properly, like git-guard does. Scripts and hooks
should become first-class citizens for uniform, unambiguous
results.

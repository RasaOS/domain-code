# TASK-005 — Linked-repo registry for multi-repo systems

**User story.** As a developer in a multi-repo system, I want to
link other repos so the project knows their location and can read
their documents.

**Why.** A project can't coordinate with sibling repos it has no
pointer to; a registry of linked repos unlocks cross-repo
awareness.

**Notes (raw idea).** Link other repos in a wider system so the
project has access to their location and can begin reading their
documents.

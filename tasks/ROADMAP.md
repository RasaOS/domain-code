# Roadmap

Phase-by-phase task registry for claude-kit's own development. Each
phase has a name, a scope paragraph, and an ordered list of tasks.
Order implies suggested ship order.

This is the kit dogfooding its own task system — see
`kit/task-rules.md`.

ID'd, but not yet assigned a phase or priority. As triage tasks
graduate, add their phase here.

See `tasks/triage/` for the current idea backlog.
